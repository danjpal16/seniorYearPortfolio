About Me

  • Hi, I’m Daniel Palagano
  • I joined the Web Design Program in my sophomore year because I’ve always been drawn to computer science
  • That year was tough, I struggled a lot and even thought about quitting
  • But instead, I stuck with it, asked Mr. Kilgore tons of questions, and kept pushing
  • My projects didn’t always look the best, but they worked, and I had fun making them
  • Beyond web design, I’m passionate about all things computers
  • I’m heading to Virginia Tech to major in Cybersecurity
  • There, I plan to work hard, land great internships, and aim for the top of my class
  • After college, I hope to land a cybersecurity role with a government agency like the NSA, FBI, CIA, or Homeland Security

Sophomore Year Coursework

  • HTML: head tags, body tags, h1 tags, p tags, etc
  • CSS: classes, ID's, padding, margin, positioning, fonts, colors, etc
  • Bootstrap: more of an advanced level of CSS, columns, rows
  • Sophomore year consisted of baseline coding, an easy itroduction to make websites

Junior Year Coursework

  • We learned JavaScript which consisted of:
  • Arrays: an ordered collection of data
  • Iterator: an object which defines a sequence and potentially a return value upon its termination
  • Variables: represents the name of a variable in a mathematical expression or a programming context

Senior Year Coursework

  • We learned more forms of JavaScript that made it easier to use and made the website look better
  • Vue: JavaScript framework for building user interfaces
  • jQuery: is used to connect with HTML elements in the browser via the DOM
  • AJAX: allows web pages to be updated asynchronously by exchanging data with a web server behind the scenes
  • JSON: a standard text-based format for representing structured data based on JavaScript object syntax
